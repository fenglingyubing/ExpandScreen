# ExpandScreen 用户文档（DOC-001）

> 本文档面向最终用户（非开发者），覆盖快速开始、安装、使用教程、FAQ、故障排查与发布说明模板。

## 1. 快速开始

### 1.1 你将获得什么

- 把 Android 设备变成 Windows 的“扩展屏幕”
- 支持 USB / WiFi 两种连接模式（取决于版本与部署方式）
- 支持触控回传（取决于设备与系统权限）

### 1.2 最短路径（推荐）

1. 在 Windows 上安装 ExpandScreen（Windows 客户端 + 驱动）
2. 在 Android 设备上安装 ExpandScreen（Android 客户端）
3. 选择连接方式：
   - **USB**：插线即可（需要 Android 开启 USB 调试）
   - **WiFi**：同一局域网（需要放行防火墙）
4. 在 Windows 中选择扩展屏幕并开始投屏

## 2. 安装说明

### 2.1 系统要求

**Windows**
- Windows 10/11（建议 WDDM 2.0+）
- .NET Desktop Runtime（若安装包未内置）
- GPU：建议支持硬件编码（NVENC / QuickSync），否则自动降级

**Android**
- Android 10+（建议 11+）
- 建议 1080p 及以上分辨率与稳定 WiFi/USB 线缆

### 2.2 Windows 客户端安装

> 若你使用“正式安装包”，按安装向导完成即可。若你使用“便携包”，按下述方式配置。

1. 解压或安装 ExpandScreen Windows 客户端
2. 首次启动时：
   - 允许防火墙弹窗（WiFi 模式需要）
   - 允许写入日志/配置（默认在用户目录）

### 2.3 Android 客户端安装

> 若你使用 Play/应用市场版本，正常安装即可；若为 APK 分发，需允许“安装未知来源应用”。

1. 在 Android 上安装 ExpandScreen
2. 首次启动时授予必要权限（网络、前台服务等）

### 2.4 驱动安装（虚拟显示器）

ExpandScreen 通过虚拟显示器驱动创建“可扩展的第二屏”。驱动安装通常随 Windows 安装包完成：

- **正式版本**：使用已签名驱动，按安装包流程完成
- **测试/开发版本**（仅供高级用户/开发者）：
  - 可能需要启用测试签名模式或安装测试证书
  - 若系统提示阻止安装，请使用官方发行包或联系维护者获取签名版本

## 3. 使用教程

### 3.1 USB 连接教程

**Android 端准备**
1. 打开“开发者选项”
2. 开启“USB 调试”
3. 使用数据线连接 Windows
4. 在弹窗中允许“USB 调试授权”

**Windows 端操作**
1. 打开 ExpandScreen
2. 选择连接方式：USB
3. 选择目标 Android 设备并连接
4. 成功后，Windows 将出现新的扩展屏幕；可在“显示设置”中调整位置/缩放

**常见提示**
- 充电线不等于数据线；请使用支持数据传输的线缆
- 部分机型需要在 USB 模式选择“文件传输/USB 共享”等

### 3.2 WiFi 连接教程

**准备**
- Windows 与 Android 连接到同一局域网（同一路由器/同一网段更稳定）
- 首次使用时允许 Windows 防火墙放行

**步骤**
1. Windows 打开 ExpandScreen，选择连接方式：WiFi
2. Android 打开 ExpandScreen，进入“发现/连接”
3. 在设备列表中选择目标设备并连接
4. 若出现配对码/确认提示，按提示完成配对

### 3.3 触控使用技巧

- 若触控定位偏移：优先检查 Windows 显示缩放（100% / 125% / 150%）与 Android 分辨率
- 若触控延迟明显：优先使用 5GHz WiFi 或 USB；在网络拥塞时降低分辨率/帧率
- 若手势不可用：部分系统手势被系统拦截，建议在应用内切换“手势/点击模式”

## 4. 常见问题（FAQ）

### 4.1 连接失败/找不到设备

- USB：确认已开启 USB 调试并授权；更换线缆/接口；尝试重新插拔
- WiFi：确认同一局域网；关闭 VPN/代理；允许防火墙；尝试重启路由器

### 4.2 画面卡顿/延迟高

- 优先切换到 USB
- WiFi 建议 5GHz，尽量靠近路由器
- 关闭占网应用（下载/投屏/网游）
- 降低分辨率或帧率（如 60 → 30）

### 4.3 没有出现“第二屏”

- 打开 Windows “显示设置”确认检测到新显示器
- 重新安装驱动或使用官方发行包
- 检查显卡驱动与系统更新

## 5. 故障排查指南

### 5.1 先做这三步（90%问题能定位）

1. 重启 ExpandScreen（Windows/Android 都重启）
2. 更换连接方式（WiFi ↔ USB）
3. 收集诊断信息并提交

### 5.2 收集诊断信息（Windows）

> 若应用提供“诊断/导出”入口，请优先使用；能自动包含版本/环境/关键日志。

- 记录：Windows 版本、显卡型号、网络环境（WiFi/USB）
- 导出日志（如有）并附上“复现步骤 + 期望结果 + 实际结果”

### 5.3 收集诊断信息（Android）

- 记录：Android 版本、机型、分辨率/刷新率
- 导出/分享应用日志（如有），或使用系统日志（logcat）协助定位
