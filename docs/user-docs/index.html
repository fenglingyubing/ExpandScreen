1. 快速开始

推荐按“最短路径”先跑通，再按需求细化画质、帧率和触控体验。

你将获得什么

  • Android 变成 Windows 扩展屏
  • USB/WiFi 连接（取决于版本与部署）
  • 触控回传（取决于设备与权限）

最短路径（推荐）

  1. 安装 Windows 客户端（含驱动）
  2. 安装 Android 客户端
  3. 选择 USB 或 WiFi 连接
  4. 在 Windows“显示设置”调整位置/缩放

2. 安装说明

系统要求

  • Windows：Windows 10/11（建议 WDDM 2.0+）
  • Android：Android 10+（建议 11+）
  • 网络：WiFi 建议 5GHz；USB 需数据线

安装提醒

  • 首次使用 WiFi 模式要允许防火墙放行
  • USB 模式需要 Android 开启 USB 调试并授权
  • 虚拟显示器依赖驱动：请优先使用官方发行包
驱动安装（虚拟显示器）

正式版本应使用已签名驱动并随安装包完成。若系统阻止驱动安装，建议改用官方发行包或联系维护者获取签名版本。

3. 使用教程

USB 连接教程

Android 端
  1. 打开“开发者选项”
  2. 开启“USB 调试”
  3. 插入数据线连接 Windows
  4. 弹窗选择“允许 USB 调试授权”
Windows 端
  1. 打开 ExpandScreen
  2. 选择连接方式：USB
  3. 选择设备并连接
  4. 在“显示设置”调整扩展屏布局
小提示：充电线不等于数据线；若设备不显示，先换线/换口，再检查 USB 模式是否为“文件传输”。

WiFi 连接教程

  • 确保 Windows 与 Android 在同一局域网（同一路由器更稳定）
  • 首次使用允许 Windows 防火墙放行
  1. Windows 选择连接方式：WiFi
  2. Android 进入“发现/连接”，选择目标设备
  3. 如出现配对码/确认提示，按提示完成配对
  4. 连接成功后，在 Windows“显示设置”调整布局
建议：WiFi 选 5GHz，尽量靠近路由器；网络拥塞时优先降低帧率（60 → 30）。

触控使用技巧

  • 触控偏移：检查 Windows 缩放与 Android 分辨率
  • 触控延迟：优先 USB，其次 5GHz WiFi
  • 手势不可用：部分系统手势会被拦截，可切换“手势/点击模式”

4. 常见问题（FAQ）

连接失败 / 找不到设备怎么办？
  • USB：确认 USB 调试已开启并授权；更换线缆/接口；重插
  • WiFi：确认同一局域网；关闭 VPN/代理；允许防火墙；重启路由器
画面卡顿或延迟很高？
  • 优先切换 USB
  • WiFi 用 5GHz，靠近路由器
  • 关闭占网应用，降低分辨率/帧率
Windows 没有出现“第二屏”？
  • 打开“显示设置”，确认检测到新显示器
  • 重新安装驱动或改用官方发行包
  • 更新显卡驱动与系统补丁

  • 应用日志或 logcat（如可获取）
